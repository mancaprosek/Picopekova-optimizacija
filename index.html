Osnove izdelave spletne strani

Priprava spletne strani

Za pripravo spletne strani je dobro poznati osnove

  • jezika HTML,
  • jezika CSS,
  • jezika JavaScript.

Gradiva in vodičev je na spletu zelo veliko. Za kratko osvežitev si lahko ogledate posnetke v YouTube kanalu Web Demystified. Bolj pregleden in podroben je vodič MDN Web Docs.

Pri pripravi datotek si lahko pomagamo z urejevalnikom Visual Studio Code. Za predogled nastajajoče spletne strani uporabimo paket Live Server ali HTML Preview.

Objava spletne strani

Spletno stran lahko objavimo s pomočjo različnih ponudnikov spletnega gostovanja. Če imamo kodo shranjeno v GitHub repozitoriju, lahko to enostavno uredimo s storitvijo GitHub Pages.

Ta spletna stran je objavljena na naslovu https://jagrdemo.github.io/spletna-stran/.

Vključevanje skript

Za dodajanje interaktivnih elementov na spletno stran se uporablja programski jezik JavaScript. Kodo, napisano v tem jeziku, vključujemo z značko <script>.

Na ta način lahko na spletno stran recimo dodamo "vljuden" gumb.

Z uporabo zunanje skriptne datoteke MathJax lahko v besedilu uporabljamo tudi $\LaTeX$ in zapišemo kakšno zanimivo formulo:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}.$$

Vključevanje zunanjih vsebin

Na spletno stran lahko vključimo tudi vsebine, ki se nahajajo na drugih spletnih naslovih. To omogoča značka <iframe> Primer je vključitev ankete, pripravljene s pomočjo Google obrazcev.

Spletna stran je bila pripravljena v okviru predavanj pri predmetu Programska oprema pri pouku (FMF, UNI-LJ) aprila 2023.
